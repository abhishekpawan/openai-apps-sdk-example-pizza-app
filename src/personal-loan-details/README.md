# Personal Loan Details Widget

This widget displays comprehensive information about Bajaj Finserv Personal Loans.

## Features

- Interest rates and loan amounts
- Tenure options
- Processing fees
- Key features and benefits
- Eligibility criteria
- Required documents
- Direct application link
- Vertical image carousel with multiple product images

## Usage

This widget is specifically designed for personal loan queries and will be called when users ask about:
- Personal loans
- PL
- Instant personal loans
- Quick personal loans
- Or any variation/misspelling related to personal loans

The AI will intelligently route personal loan queries to this dedicated widget.
