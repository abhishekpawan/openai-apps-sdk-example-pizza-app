# Gold Loan Details Widget

This widget displays comprehensive information about Bajaj Finserv Gold Loans.

## Features

- Competitive interest rates starting from 9.5% p.a.
- Loan amount: ₹5,000 - ₹2 Crore
- Flexible tenure: 3 - 36 months
- Key features including zero prepayment charges
- Eligibility criteria
- Required documents
- Direct application link
- Vertical image carousel with multiple product images

## Usage

This widget is specifically designed for gold loan queries and will be called when users ask about:
- Gold loans
- Loan against gold
- Gold jewelry loans
- Or any variation/misspelling related to gold loans

The AI will intelligently route gold loan queries to this dedicated widget.
